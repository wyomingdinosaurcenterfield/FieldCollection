Setup Guide
No Google setup needed: Use fully offline. Export CSV anytime to import into Sheets manually.

Full Google sync (one-time setup):
1. Create a Google Service Account in console.cloud.google.com → Enable Drive + Sheets APIs
2. Deploy cloudflare-worker.js to Cloudflare Workers, add the service account JSON as the SA_CREDENTIALS secret
3. Share your Google Sheet and Drive folder with the service account email
4. Paste the Worker URL, Sheet ID, and Folder ID above
5. Entries sync automatically — no sign-in ever required

Hosting tip: Upload this HTML to GitHub Pages. All team members share the same URL, each sets their own name above.
